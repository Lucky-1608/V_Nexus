# V_Nexus

Your all-in-one personal operating system. Master your habits, tasks, finances, and knowledge in one central hub.

## üåü Key Features & Modules

### 1. **Dashboard**
- **Central Hub**: High-level overview of your productivity and finances.
- **Widgets**: Real-time stats for daily habits, tasks, finance summaries, and goal progress.

### 2. **Habits**
- **Tracking**: Create and track daily habits with frequency settings.
- **Analytics**: Visual graphs of completion rates and consistency streaks.
- **Workflow**: Create a habit -> Mark as done daily -> Watch stats update in real-time.

### 3. **Tasks**
- **Management**: efficient task manager with priorities and status tracking.
- **Organization**: Filter tasks by status (Todo, In Progress, Done).

### 4. **Finances**
- **Transaction Tracking**: Log income and expenses with detailed categories.
- **Projects**: Link financial activities to specific projects.
- **Analytics**: Track total balance and expenditure trends.

### 5. **Resources & Knowledge**
- **Library**: Digital asset management for links, books, and videos.
- **Notes**: Integrated note-taking system.
- **Collections**: Organize resources into custom collections.

### 6. **Learning & Growth**
- **Learning Paths**: Structured guides for skill acquisition.
- **Roadmaps**: Visual timeline planning for long-term objectives.
- **Goals**: Set and track specific milestones.
